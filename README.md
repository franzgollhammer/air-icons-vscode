# Air File Icons

A clean, minimal file icon theme for Visual Studio Code inspired by the JetBrains New UI icon set.

## Features

- 112 file type icons
- Dark and light theme variants
- Covers all major languages and frameworks

## Installation

1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X`)
3. Search for "Air File Icons"
4. Click Install
5. Open Command Palette (`Cmd+Shift+P`) > "Preferences: File Icon Theme" > "Air File Icons"

## Supported File Types

Includes icons for: JavaScript, TypeScript, React, Vue, Svelte, Python, Rust, Go, Java, Kotlin, Swift, C/C++, C#, Ruby, PHP, Dart, Elixir, Haskell, Scala, and many more.

Also recognizes common config files like Docker, Git, Webpack, ESLint, Prettier, and package managers.
